# Spin Rhythm

A Roblox game managed by [pesde](https://pesde.dev/) and synced with [Rojo](https://rojo.space/).

## Authors

- [josephh310](https://github.com/josephh310)
- [lonegladiator](https://github.com/Lonegwadiwaitor)

## Getting Started

**1. Install dependencies**

```sh
pesde install
```

**2. Start the Rojo dev server**

```sh
rojo serve
```

**3. Connect Roblox Studio**

Open your place file in Roblox Studio and connect to the local Rojo server via the Rojo plugin.
